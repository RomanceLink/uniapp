# Dataset Spec

## 标注格式

每行一条：

```text
相对图片路径<TAB>标签文本
```

例如：

```text
images/led/scale_0001.jpg	0.23|989.4
images/water_meter/meter_0001.jpg	462796.7589
```

## 标签约定

- 电子称：
  如果一张图中存在两组主要数字，按上到下顺序用 `|` 连接
- 水表：
  直接写最终完整读数
- 只允许字符：
  `0-9 . - |`

## 拍摄要求

- 同一设备至少覆盖：
  - 正拍
  - 轻微倾斜
  - 反光
  - 弱光
  - 模糊
- 每种设备最好 30 张以上

## 不建议混入

- 完全看不清数字的图片
- 后期严重压缩的转发图
- 带太大遮挡的图
